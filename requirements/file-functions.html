2.2. File Functions

These introductory scenarios show an individual user adding various basic items to a spreadsheet. Adding items is an essential operation for setting up a spreadsheet for a class. In these scenarios, we assume that the user has just been registered in the grading system and so has a brand new spreadsheet with no entries. The scenarios will cover the user creating a new class spreadsheet, adding a single student to the class without importing a roster, adding new categories of grades, and finally adding individual assignments within those categories.

The scenarios in this section show a user adding a variety of items to the Grader Tool spreadsheet, including

  • creating a new class spreadsheet
  • adding a new row of student data
  • inserting a new column for gradable categories
  • adding individual gradable assignments to a category
