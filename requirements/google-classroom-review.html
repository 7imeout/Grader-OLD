1.5.4. Google Classroom

Google Classroom builds on the popular Google Apps to create classroom management software. It manages assignments, and grades from a web app.

Good Features:

  • Manages multiple classes for instructors and students
  • Student roster data simple to populate
  • Easy to add new assignments
  • Manages student submissions
  • Integrates with other google products
  • Students can see assignments from their dashboard
  • Students can see their graded assignments
  • Everything is synchronized with other google services
  • Google will not charge Cal Poly any money

Bad Features:

  • no way to modify or enhance software
  • poor integration of file types not supported by Google Apps
  • students must have a Google account
  • instructors must have a Google account
  • limited functionality

Missing Features:

  • Spreadsheet style input interface that is enhanced for grading
  • Curve adjustments
  • Student centered what if analysis
  • Integration with Cal Poly’s authentication system
  • A way for multiple instructors to grade for a single class
  • A way to post grades all at once
  • A way to see how other students scored
