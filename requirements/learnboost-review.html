1.5.5. LearnBoost

LearnBoost is a new grading program for teachers and students that brings a fresh new feel to the way grading programs look and act. LearnBoost is a web based grading application that provides teachers the functionality to input grades easily, add students to certain classes, and examine different graphs of student progression throughout the course.

Overall, LearnBoost provides a range of grading tools to the user, but also sacrifices some of its functionality for ease of use and presentation to the user.

Good Features:

  • clean, sharp, modern user interface
  • all information is stored on a cloud network
  • all basic grading tool functionality is provided
  • able to import CSV file for student roster
  • button for excused work
  • ability to sort roster

Bad Features:

  • changing the weights of different sections is confusing
  • assignments lacked organization
  • graphs were too static
  • changing the grade distribution took a lot of work
  • certain features are not obvious i.e printing and exporting to CSV

Missing Features:

  • student handin functionality
  • student grade predictor
  • team grading functionality
  • late policy option
  • can not have more than one class open at a time (no multiple tabs)
